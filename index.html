Visualizza tutti i tuoi dati di salute — passi, frequenza cardiaca, sonno, calorie, peso e molto altro — direttamente nel browser, senza server esterni.

01

Crea un progetto su Google Cloud

Vai su console.cloud.google.com → Nuovo progetto → assegna un nome (es. "FitDash").

02

Abilita la Fitness API

Nel pannello del progetto → API e servizi → Libreria → cerca Fitness API → Abilita.

03

Configura la Schermata consenso OAuth

API e servizi → Schermata consenso OAuth → tipo Esterno → compila nome e email → aggiungi il tuo account come Utente di test.

04

Crea credenziali OAuth 2.0

API e servizi → Credenziali → Crea credenziali → ID client OAuth.
Tipo: Applicazione web.
Origini autorizzate: aggiungi http://localhost e null (per apertura da file).
Copia il Client ID e incollalo qui sotto.
